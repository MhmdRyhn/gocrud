Creating Author:
================
```
{
    "on": "author",
    "apply": "create",
    "name": "Daron Acemoglu",
    "email": "daron@mail.com",
    "phone": "+123456789",
    "age": 45,
    "address": "Turkey"
}


{
    "on": "author",
    "apply": "create",
    "name": "Joe Navarro",
    "email": "joe@mail.com",
    "phone": "+102456489",
    "age": 52,
    "address": "USA"
}
```


Get all author
===============
```
{
    "on": "author",
    "apply": "fetch_all"
}
```


Get single author
==================
```
{
    "on": "author",
    "apply": "fetch_one",
    "id": 1
}
```


Update Author
==============
```
{
    "on": "author",
    "apply": "update"
}
```
